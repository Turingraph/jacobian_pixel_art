# jacobian_pixel_art

Jacobian Pixel Art is the pixel art maker that allow use to transform grids according to user defined 2 variables non linear PDE that looks like a linear grid, a.k.a. Euclidean grid locally. The default grid is identical matrix, which is identical to typical pixel art. User can color those grids similar to Pixel art.
