# 2 Types of folder

1.	Building Block Folder
*	This folder contains only files that do not import any other JSX Element from other folders, except `str/` and `type/`.
*	`atom/, str/, type/`
2.	Composible Folder
*	This folder contains only files that import other JSX Element from other folders.
*	Every folder inside `ui/` dir that is not `atom/, str/, and type/`.
